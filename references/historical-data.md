## 各类别最优内容结构

### 竞赛成果类
> 奖项/数据前置 → 1-2句引子 → 具体项目/比赛介绍 → 学生/教师采访引语 → 过程故事(1-2个细节) → 未来展望

### 校园活动类
> 趣味标题+导语 → 活动背景 → 现场描述(图文穿插) → 学生/参与者采访 → 活动意义总结

### 教学创新类(枝乎)
> 问题提出 → 教学背景/痛点 → 具体策略/方法框架 → 数据/案例验证 → 方法论提炼

### 升学成果类
> 核心数据前置(录取数/学校名) → 细分榜单 → 典型案例故事 → 师生感言 → 历史成绩对比
